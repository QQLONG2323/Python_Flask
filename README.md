# 將python專案建立EXE檔: 
1. 安裝PyInstaller：  
使用pip在命令提示字元中輸入以下命令：  
pip install pyinstaller

2. 創建spec文件：  
在命令提示字元中，導航到包含您的Python檔案的目錄，然後輸入以下命令：  
pyinstaller your_script_name.py

3. 創建執行檔：  
在命令提示字元中，輸入以下命令以創建執行檔：  
pyinstaller your_script_name.spec

4. 在 dist 文件夾中找到可執行文件並運行即可
